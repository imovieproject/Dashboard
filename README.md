# Dashboard
![](logo.png)
iMovie是一个电影推荐平台。用户可以在该网站上搜索、浏览电影，并且能够获得根据用户口味推荐的电影。

## 1.0.0 Features
- 浏览、搜索电影
- 给电影打分
- 获得推荐的电影
